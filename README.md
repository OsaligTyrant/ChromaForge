# ChromaForge

ChromaForge is a desktop tool that batch-edits PNGs, groups outputs by filename
prefix, and builds sprite sheets and tilemaps with a simple GUI (non-matching files go to `needs_sorting`). It also includes a manual layout editor.

Version: Beta 2.1

## Quick Start
- Launch `ChromaForge.exe` in this folder.
- Update folders, colors, and modes in the app UI.
